# 改ページの自動計算のメモ

## 用紙サイズ

A4なので「210x297mm」

## 余白

上下左右ともに1インチ（25.4mm）

## 印刷領域

297 - 25.4 * 2 = 246.2mm

## ポイント・ピクセル・ミリメートルの対応

### ポイントとミリメートルの対応

- Excel の行の高さはポイントで指定する
- DTP や印刷では「1インチ = 72ポイント」が標準となっている
- 1ポイント = 0.0138888889インチ ≒ 0.35277777777778mm ≒ 0.35mm

### ピクセルとミリメートルの対応

- Windows の論理DPIは標準で 96dpi となっている
- 1インチ = 96ピクセル（Windowsの論理96dpi）
- 1ピクセル = 0.01041666666667インチ = 0.26458333333333mm ≒ 0.265mm

- 画面の解像度スケールを 125%・150% にすると DPI が変わるが、Excel は内部に独自の DPI (96dpi) を持っているため、実際は変化無し。

### ポイントとピクセルの対応

- 1pt = 96/72 px = 4/3 px ≈ 1.3333px
- 1px = 72/96 pt = 3/4 pt = 0.75pt
- Excelでの具体例

| height at line (pt) | point to pixel |
|---------------------|----------------|
| 15 pt (default)     | 20 px          |
| 13.5 pt             | 18 px          |
| 12 pt               | 16 px          |
| 0.75 pt             | 1 px           |

- 1inch = 25.4mm ≒ 96pixel
- 1mm ≒ 約3.78pixel

## タイトル行の高さ

15ポイント * 0.35mm/ポイント ≒ 5.25mm

## 空白行の高さ

15ポイント * 0.35mm/ポイント ≒ 5.25mm

## タイトル行を除いた印刷領域

246.2 - 5.25 = 240.95mm ≒ 241mm
